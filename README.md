# 简介
Unity读取Excel工具，点击F8自动生成C#索引字段，能读取Excel导出的二进制文件，运行时读取Excel文件，支持Win/Android/iOS/Mac/Linux

## 使用流程
导入插件，在目录StreamingAssets/config下创建Excel后，点击菜单栏->开发工具->导入配置表_F8  
游戏运行时读取Excel数据，点击菜单栏->开发工具->运行时读取Excel_F7
